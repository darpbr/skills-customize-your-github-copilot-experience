# 📘 Assignment: Statistics with Python

## 🎯 Objective

Learn to perform statistical calculations and data analysis using pandas and numpy libraries. Students will analyze datasets, compute descriptive statistics, and perform basic statistical tests.

## 📝 Tasks

### 🛠️	Basic Statistical Analysis

#### Description
Load a dataset using pandas and compute basic descriptive statistics including mean, median, standard deviation, and quartiles for numerical columns.

#### Requirements
Completed program should:

- Load data from a CSV file using pandas
- Display basic information about the dataset (shape, columns, data types)
- Calculate and display descriptive statistics for numerical columns
- Handle missing values appropriately
- Use numpy for any additional calculations


### 🛠️	Statistical Calculations and Tests

#### Description
Perform statistical calculations such as correlation analysis, hypothesis testing, and data grouping using pandas and numpy.

#### Requirements
Completed program should:

- Compute correlation matrix for numerical variables
- Perform basic statistical tests (e.g., t-test or chi-square if applicable)
- Group data by categorical variables and compute summary statistics
- Create visualizations to support statistical findings
- Document findings and interpretations in comments
