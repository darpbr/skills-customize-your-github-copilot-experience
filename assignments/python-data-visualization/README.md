# 📘 Assignment: Python Data Visualization

## 🎯 Objective

Learn to create charts and graphs in Python using `matplotlib` and `plotly` to explore and communicate data insights visually.

## 📝 Tasks

### 🛠️ Plot with Matplotlib

#### Description
Build a script that reads sample sales data and generates two charts using `matplotlib`.

#### Requirements
Completed program should:

- Load a list of monthly sales values from the provided data structure or CSV.
- Plot a line chart of sales over months with labels, title, and legend.
- Plot a bar chart of sales by month with axis labels.
- Save both charts as PNG files (`sales_line.png`, `sales_bar.png`).

### 🛠️ Interactive Chart with Plotly

#### Description
Extend the script to create an interactive visualization using `plotly`.

#### Requirements
Completed program should:

- Use the same sales data to generate an interactive `plotly` line or bar chart.
- Include hover tooltips showing month and sales.
- Save the figure as an HTML file (`sales_interactive.html`).

### 🛠️ Data Analysis Summary

#### Description
Write a function to compute basic summary statistics for the sales data.

#### Requirements
Completed program should:

- Compute and print total sales, average sales, highest month, and lowest month.
- Display the results clearly in the console output.
