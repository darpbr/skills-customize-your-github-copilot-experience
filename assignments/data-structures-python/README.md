# 📘 Assignment: Data Structures in Python

## 🎯 Objective

Learn and practice using Python's built-in data structures: lists, dictionaries, sets, and tuples. You will implement various operations and understand how to manipulate these data types effectively.

## 📝 Tasks

### 🛠️	Lists Operations

#### Description
Create a Python program that demonstrates various list operations. You'll need to create functions that perform common list manipulations.

#### Requirements
Completed program should:

- Create an empty list and add elements using append() and extend()
- Remove elements using remove(), pop(), and del
- Sort the list in ascending and descending order
- Find the index of a specific element
- Create a list comprehension to filter even numbers from a range

### 🛠️	Dictionary Manipulation

#### Description
Implement a program that uses dictionaries to store and manage student information. Create functions to add, update, and retrieve data.

#### Requirements
Completed program should:

- Create a dictionary with student names as keys and grades as values
- Add new students and update existing grades
- Remove students from the dictionary
- Iterate through the dictionary to display all students and their grades
- Use dictionary methods like keys(), values(), and items()

### 🛠️	Sets and Tuples

#### Description
Write a program that demonstrates the use of sets and tuples. Show how sets can be used for unique collections and tuples for immutable data.

#### Requirements
Completed program should:

- Create sets and perform union, intersection, and difference operations
- Add and remove elements from a set
- Create tuples and access elements by index
- Demonstrate that tuples are immutable (cannot be changed after creation)
- Convert between lists, sets, and tuples
